# Homebrew Tap Nexo-Fortis 🍺

Este é o repositório oficial de distribuição das ferramentas da Nexo-Fortis via Homebrew.

## 🚀 Como usar

Adicione este tap ao seu sistema:

```bash
brew tap fortissolucoescontato-bit/homebrew-tap
```

## 📦 Fórmulas Disponíveis

| Ferramenta | Descrição | Comando de Instalação |
|------------|-----------|-----------------------|
| **Kortex** | Orquestrador principal de agentes de IA | `brew install kortex` |
| **Kortex-Engram** | Motor de memória persistente | `brew install kortex-engram` |
| **Carbon Guardian Angel** | Guardião de diretrizes e persona | `brew install carbon-guardian-angel` |

## 🛠️ Atualização

Para atualizar suas ferramentas instaladas para a versão mais recente:

```bash
brew update
brew upgrade <formula>
```
